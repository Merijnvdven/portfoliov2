3. Keybinds

Wat heb ik gedaan?

Dit is een simpele POC waar ik laat zien dat ik keybinds kan toepassen. Dat houdt in dat ik een bepaalde actie kan koppelen aan een toets op mijn toetsenbord. Nu heb ik in css een balletje gemaakt die je met de pijltjes kan besturen. Bij verandering van elke richting veranderd de kleur van het balletje





Wat heb ik geleerd?

Ik heb geleerd hoe ik keybinds moet gebruiken in javascript. Ik weet nu hoe ik een object moet verplaatsen aan de hand van een paar toetsen. Als extra heb ik verschillende kleuren toegevoegd bij elke richting. Dus beweeg je de bal naar rechts wordt het balletje bijvoorbeeld blauw. Dus ik heb 2 verschillende acties op een keybind gezet





Wat kon beter?

Het balletje beweegt een stukje elke keer als ik een pijltje indruk. Het kan vloeiender door het balletje te laten bewegen als ik een knop ingedrukt houdt. Ook valt het balletje op dit moment nog buiten het beeld(als je hem daarheen stuurt).
