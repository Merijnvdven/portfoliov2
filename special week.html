Special Week

Wat heb ik gedaan?

De opdracht hier was om samen in tweetallen te verdiepen in een javascript framework en het daarna in de praktijk toe te kunnen passen. Ik heb samen met een andere student gekozen voor "vue". We hebben deze keuze gemaakt omdat het handig op te pakken is voor beginners. We hebben vue gebruikt om een kleine website (in dit geval soort domino's)neer te zetten met een paar functionaliteiten zoals pizza's toevoegen en bestellen.





Wat heb ik geleerd?

Ik heb geleerd hoe je overweg kan met een javascript framework (vue). Ik heb hierbij geleerd hoe je een systeem in elkaar zet om objecten (in dit geval piza's) toe voegen, te verwijderen en het totaal optelt. Ook heb ik gewerkt met een dropdown menu om de grootte van de pizza te kunnen kiezen. Omdat ik deze opdracht niet alleen heb gedaan, maar ook met iemand anders heb ik het deel wat niet door mij is gemaakt weggeknipt.





Wat kon beter?

Meer functies toevoegen aan de site. Zoals bezorggegevens van de klant zodat de pizza ook echt besteld kan worden. Als je een pizza verwijderd uit je bestelling kan je ook onder de 0 pizza's uitkomen. daar had ik achteraf langer naar kunnen kijken om dat op te lossen.
