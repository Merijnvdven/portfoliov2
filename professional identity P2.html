Wat heb ik geleerd?

Na het maken van dit spel weet ik nu meer van javascript af. Om specifieker te zijn Ik heb met dit spel geleerd hoe ik moet werken met keybinds,if en else statements, objecten inladen en de score laten optellen. Dit zijn dingen die waarschijnlijk nog veel vaker terugkomen later in de opleiding. Vooral de if en else statements zijn van belang. Ook als je iets anders maakt dan een minigame





Wat kon beter?

Ik heb voor dit spel redelijk veel javascript code moeten gebruiken. Ik had vast manieren kunnen vinden om dit in minder regels code te kunnen doen. Ik had overzichterlijker kunnen werken in visual studio code. Bijvoorbeeld de code meer in groepjes neer te zetten en daarbij omschrijven wat die specifieke code doet binnen het spel.
